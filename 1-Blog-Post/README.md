# HTML Blog Post Project  

## Description  
This project is my first hands-on implementation of HTML, where I created a simple blog post webpage. The goal was to apply the foundational concepts of HTML by structuring content in a clear, semantic, and readable way.

## Objective  
The main objective of this project was to understand how to use HTML to build the structure of a webpage. I focused on organizing text content, adding links, images, and using appropriate HTML elements to represent a blog post.

## Features  
- Structured blog layout using semantic HTML elements  
- Headings and paragraphs for clear content hierarchy  
- Embedded images to enhance the article  
- Hyperlinks for navigation and references  
- Lists for organizing information  

## Technologies Used  
- HTML5  

## What I Learned  
Through this project, I gained a solid understanding of:  
- Basic HTML document structure (`<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`)  
- Using semantic tags such as `<header>`, `<article>`, `<section>`, and `<footer>`  
- Creating headings, paragraphs, and lists  
- Adding images and links  
- Writing clean and readable HTML code
- Debugging using devtools
